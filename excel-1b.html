1.26 — 10 pts

Column widths + wrap text + Heading 4

  1. Drag-select column headers B through F.
  2. Home → Cells → Format → Column Width → 13 → OK.
  3. Select B4:F4 (the header row).
  4. Home → Alignment → Wrap Text.
  5. With B4:F4 still selected: click Center (horizontal) then Middle Align (vertical) in the Alignment group.
  6. Cell Styles → Heading 4.
