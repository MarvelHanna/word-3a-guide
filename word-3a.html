3.04 — part of 10 pts

Footnote MLA style

  1. Control-click in any footnote text → Style.
  2. Footnote Text is already selected. Click Modify.
  3. Set Font Size to 11.
  4. Lower-left: Format → Paragraph.
  5. Special: First line (By defaults to 0.5").
  6. Line spacing: Double.
  7. OK → OK → Apply.

  3. Pages: 5 → OK.

Insert and format Works Cited

  1. Cursor still on the empty Page 4 paragraph. Confirm References → Style is MLA.
  2. References → Citations & Bibliography → Bibliography → Works Cited (near bottom).
  3. Drag-select the Works Cited heading AND all three citation entries.
  4. Format → Paragraph → Before: 0, After: 0, Line spacing: Double → OK.
  5. Triple-click the Works Cited heading to select only it.
  6. Set Font: Aptos (Body), Size: 11, Font Color arrow → Theme Colors → column 2, row 1.
  7. Cmd+E to center.
