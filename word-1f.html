Step 2 — 6 pts

File Name in footer

  1. Insert → Header & Footer → Footer → Edit Footer
  2. Mac: Insert → Field → Document Information → FileName → OK
  3. Close Header and Footer
Must be the FileName field, not typed text.
Step 3 — 10 pts

Line spacing 1.5

  1. Cmd+A to select all.
  2. Home → Paragraph → Line and Paragraph Spacing → 1.5
Step 4 — 11 pts

Center title + size 24

  1. Triple-click the title Light Magic Studios to select the line.
  2. Cmd+E to center.
  3. Set Font Size to 24.
Step 5 — 6 pts

Top/bottom margins 0.5"

  1. Layout → Page Setup → Margins → Custom Margins
  2. Top: 0.5", Bottom: 0.5". Leave left/right alone.
  3. OK.
Step 6 — 10 pts

First-line indent (3 paragraphs)

  1. Click in the first paragraph below the picture. Shift-click at the end of the third paragraph below the picture to select all three.
  2. Format → Paragraph → Indents and Spacing → Special: First line (defaults 0.5") → OK.
Step 7 — 6 pts

Spacing Before/After 6 pt

  1. Cmd+A to select all.
  2. Layout → Paragraph: Before: 6 pt, After: 6 pt.
Step 8 — 11 pts

Square bullets + right tab w/ dot leaders

  1. Drag-select the three paragraphs starting with Small Size Add-ons.
  2. Home → Paragraph → Bullets arrow → Bullet Library → filled square.
  3. With the list still selected: Format → Tabs.
  4. Tab stop position: 6". Alignment: Right. Leader: 2 .... (dot leader). Click Set → OK.
Step 9 — 11 pts

Numbered list

  1. Find the paragraph beginning Light Magic Studios can provide. Click at the very end (after the colon).
  2. Press Return.
  3. Format → Paragraph → Indentation → Special: (none) → OK.
  4. Home → Paragraph → Numbering (default 1, 2, 3 style).
  5. Type each item, Return between (no Return after the third):
Earn 5% off of every order
Free preview photos
Photo packages and add-ons
Step 10 — 11 pts

Insert Equation SmartArt, size 1 × 6.5

  1. Cmd+Fn+Right to end of document. Click after the period.
  2. Insert → SmartArt → Process category → Equation (circles joined by + and =). OK.
  3. Click the outside border of the SmartArt to select the whole thing.
  4. In Size: Height = 1", Width = 6.5".
Step 11 — 6 pts

Wrap + position

  1. SmartArt selected: Shape Format → Arrange → Wrap Text → Square.
  2. Position → More Layout Options.
  3. Horizontal: Alignment Centered relative to Page.
  4. Vertical: Alignment Bottom relative to Margin.
  5. OK.
Step 12 — 6 pts

Text in the three circles

  1. Click the first circle. Type or paste:
Quality
  1. Click the second circle:
Service
  1. Click the third circle:
Light Magic
Step 13 — 6 pts

Color + 3-D Polished style

  1. Click the SmartArt.
  2. SmartArt Design → Change Colors. Under Colorful, pick Colorful Range – Accent Colors 4 to 5 (hover tooltips).
  3. SmartArt Design → SmartArt Styles gallery. Under 3-D, pick Polished (first 3-D style).
Done
