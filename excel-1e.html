Step 2 — 5 pts

Vapor Trail theme

  1. Page Layout → Themes → Themes. Click Vapor Trail.
  2. Not there? Browse for Themes… → pick Vapor Trail.thmx from the assignment folder.
Step 3 — 6 pts

Fill May and June (B3 → D3)

  1. Click B3 (should already say April).
  2. Drag the fill handle right to D3. C3 fills as May, D3 as June.
Step 4 — 12 pts

Merge, center, style

  1. Select A1:F1. Home → Merge & Center. With merged A1 active: Cell Styles → Title.
  2. Select A2:F2. Merge & Center. Cell Styles → Heading 1.
  3. Select B3:F3. Home → Alignment → Center.
Step 5 — 6 pts

Widths + missing data

  1. Click column A header. Home → Cells → Format → Column Width → 22 → OK.
  2. Drag-select columns B through F. Format → Column Width → 16 → OK.
  3. In row 7 (the row missing data), enter:
  • B7: 112727.85
  • C7: 121691.64
  • D7: 128964.64
Step 6 — 9 pts

AutoSum row 8 + column E

  1. Click B8. Press Cmd+Shift+T (AutoSum). Confirm on the Formula Bar checkmark.
  2. Click B8 again. Drag fill handle right to D8.
  3. Click E4. Press Cmd+Shift+T. Confirm.
  4. Click E4 again. Drag fill handle down to E8.
Step 7 — 8 pts

Heading 4 + Total + AutoFit

  1. Select B3:F3, Cmd-click A4:A7. Cell Styles → Heading 4.
  2. Select B8:E8. Cell Styles → Total.
  3. Click column A header. Home → Cells → Format → AutoFit Column Width.
Step 8 — 8 pts

Number formats

  1. Select B4:E4 and Cmd-click B8:E8. Home → Number → Accounting Number Format ($).
  2. Select B5:E7. Home → Number → Comma Style (,).
Step 9 — 10 pts

Insert chart

  1. Select A3:D7 (no totals).
  2. Insert → Charts → Recommended Charts.
  3. Pick the Clustered Column chart with months along the bottom and product names in the legend. OK.
Step 10 — 5 pts

Position chart at A10

  1. Drag the chart so its upper-left corner sits in cell A10, approximately under "and" in row 7.
Step 11 — 5 pts

Chart Style 13 + colors + title

  1. With the chart selected: Chart Design → Chart Styles → Style 13 (hover tooltips to confirm).
  2. Chart Design → Change Colors → Colorful → second row.
  3. Click the chart title placeholder. Paste:
2nd Quarter Pet Product Sales
Step 12 — 10 pts

Sparklines + markers + style

  1. Select B4:D7.
  2. Insert → Sparklines → Line.
  3. Location Range: F4:F7. OK.
  4. With F4:F7 selected: Sparkline → Show → Markers ON.
  5. Sparkline → Style gallery → first row, second style.
Step 13 — 8 pts

Center horizontally + footer

  1. Click A1. Page Layout → Page Setup → Margins → Custom Margins.
  2. Margins tab → check Horizontally under "Center on page".
  3. Header/Footer tab → Custom Footer. Left section → click Insert File Name icon. &[File] appears. OK → OK.
Step 14 — 8 pts

Landscape + properties

  1. File → Print. Settings → switch Portrait to Landscape Orientation. Esc out.
  2. Apple menu: File → Properties → Summary. Fill:
pet product sales
CAS 50, #ON1
Marvel Hanna
  1. OK.
Done
