1.01–1.02 — 10 pts

Title, subtitle, row labels

  1. Click A1. Type and press Return after each:
The Easy Pantry
Third Quarter Sales for Fruit Spreads
  1. In the Name Box (top-left), type a4 and press Return to jump there.
  2. Type each row label, Return between each:
  • A4: Jams
  • A5: Jellies
  • A6: Preserves
  • A7: Spiced Chutneys
  • A8: Total
1.03 — 10 pts

Month headers (AutoFill)

  1. Click B3. Type July and press the checkmark in the Formula Bar (keeps B3 active).
  2. Hover the lower-right corner of B3 until you see the + fill handle.
  3. Drag right to D3. Release. C3 fills as August, D3 as September.
1.04 — 5 pts

Widen column A + center headers

  1. Drag the line between column A and column B headers to the right until column A is a bit wider than Spiced Chutneys.
  2. Select range B3:D3.
  3. Home → Alignment → Center.

1.06 — 4 pts

Total row formulas

  1. Click B8. Type =SUM(B4:B7) and press Return. (Result: 204557.97)
  2. Click B8 again, drag the fill handle right to D8. Excel adjusts to =SUM(C4:C7) and =SUM(D4:D7).
1.07 — 5.2 pts

Total column + Trend header

  1. E3: type Total, Return.
  2. E4: press Cmd+Shift+T (AutoSum). Excel proposes =SUM(B4:D4). Click the Formula Bar checkmark.
  3. With E4 active, drag the fill handle down through E8.
  4. Click F3, type Trend, Return.
1.08 — 10 pts

Merge & Center + Cell Styles

  1. Select A1:F1. Home → Alignment → Merge & Center. With merged A1 still active: Cell Styles → Title.
  2. Select A2:F2. Merge & Center. Then Cell Styles → Heading 1.
  3. Select B3:F3, then Cmd-click to also select A4:A8 (both ranges).
  4. Cell Styles → Heading 4.
1.09 — 16 pts

Number formats (biggest single block)

  1. In the Name Box, type b4:e4,b8:e8 and press Return. Both ranges selected.
  2. Home → Number → Accounting Number Format ($).
  3. Select B5:E7. Home → Number → Comma Style (,).
  4. Select B8:E8. Cell Styles → Total (single top border + double bottom border).
1.10 — 5 pts

Slice theme + AutoFit

  1. Page Layout → Themes → Themes. Click Slice.
  2. If Slice isn't there: Browse for Themes… → navigate to the assignment folder → pick Slice.thmx.
  3. Click the A column header to select column A.
  4. Home → Cells → Format → AutoFit Column Width.
  5. If anything still shows ####, select those columns and AutoFit too.
1.11 — 3 pts

Insert Clustered Column chart

  1. Select A3:D7 (no totals).
  2. Insert → Charts → Recommended Charts.
  3. Pick the Clustered Column chart that shows product names (Jams, Jellies, Preserves, Spiced Chutneys) along the bottom. OK.
1.12 — 7.8 pts

Chart style, color, title, position

  1. Click the chart title placeholder above the chart. Type or paste the title:
Third Quarter Sales for Fruit Spreads
  1. With the chart selected: Chart Design → Chart Styles → Style 6 (scroll mid-gallery, hover tooltips).
  2. Chart Design → Change Colors → Colorful Palette 3 (3rd row under Colorful).
  3. Drag the chart so its upper-left corner lands in cell A10.
  4. Click outside the chart to deselect.
1.13 — 8 pts

Line sparklines with markers

  1. Select B4:D7.
  2. Insert → Sparklines → Line.
  3. In the dialog, click the Location Range box and select cells F4:F7 (or type F4:F7). OK.
  4. With F4:F7 still active: Sparkline → Show → Markers checkbox on.
1.14 — 5 pts

Footer + center horizontally

  1. Click A1. Page Layout → Page Setup → Margins → Custom Margins.
  2. Margins tab → check Horizontally under "Center on page".
  3. Switch to Header/Footer tab → Custom Footer.
  4. In the Left section, click the Insert File Name button (sheet-of-paper icon). &[File] appears. OK.
  5. OK on Page Setup.
